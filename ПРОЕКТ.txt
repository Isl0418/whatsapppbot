  WHATSAPP BOT — ТЕХПОДДЕРЖКА (ГОТОВЫЙ ПРОЕКТ)
═══════════════════════════════════════════════════════════════

ЭТО И ЕСТЬ ПРОЕКТ. Вся папка «whatsappbot» — готовый код.

Где лежит на компе:
  c:\Users\user\PycharmProjects\whatsappbot

Что внутри:
  run.py              — запуск бота (точка входа)
  requirements.txt    — зависимости (aiohttp, openai, python-dotenv)
  .env.example        — образец настроек (скопируй в .env и заполни)
  .gitignore          — что не заливать в Git
  README.md            — описание на английском
  ПРОЕКТ.txt          — этот файл

  bot/                — код бота
    config.py         — настройки и переменные окружения
    logger.py         — логи
    state.py          — состояние (вкл/выкл, очереди)
    instructions.py   — база инструкций и поиск по ключам
    green_api.py      — работа с Green API (WhatsApp)
    handlers.py       — обработка сообщений и кнопок
    main.py           — главный цикл
    ai/
      vision.py       — разбор фото (EasyOCR + OpenAI)
      deepseek.py     — разбор текста (DeepSeek)

  logs/               — появятся после первого запуска
  temp_files/
  photos/             — сюда положи картинки для инструкций (напр. модульпечати.jpg)

Первый запуск:
  1. Открой терминал в этой папке.
  2. Активируй окружение:  .venv\Scripts\activate
  3. Установи зависимости:  pip install -r requirements.txt
  4. Скопируй настройки:    copy .env.example .env
  5. Открой .env и впиши свои ключи (Green API, DeepSeek, OpenAI).
  6. Запуск:  python run.py

Старый однострочный бот (whatsappbot.py) оставлен для сравнения;
рабочий код — в папке bot/ и запускается через run.py.
